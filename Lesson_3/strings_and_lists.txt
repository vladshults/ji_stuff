if "ab" in "abcd":

Рекурсия - вызов функцией самой себя. Пример - вычисление факториала
База рекурсии - место (значение), где вычисления остановятся. Нет базы - будет бесконечный цикл

def factorial(n):
    if n < 0 or not isinstance(n, int):
        return None

    if n in (0, 1):
        return 1
    return n * factorial(n - 1)
